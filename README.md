# simple-project-planner

## Project setup

```
npm install
```

### Running the project / branch

#### Start JSON Server

```
npx json-server --watch ./data/db.json
```

#### Then compile and hot-reload the Vue app

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
